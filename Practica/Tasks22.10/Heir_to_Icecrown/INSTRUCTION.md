# Рыцарь Смерти и Дракон Плети  
**Текстовая RPG в мире World of Warcraft: Wrath of the Lich King**

После падения Короля-лича ты — рыцарь Альянса, отправленный исследовать Ледяную Корону.  
Но древние силы не умирают... Они ждут **нового наследника**.

Сразись с остатками Плети, найди меч Артаса и реши судьбу воскрешённого дракона Сарторакса.  
Твой выбор определит: станешь ли ты **щитом**, **жертвой** или **властелином Тьмы**?

---

## ▶️ Как запустить игру

1. **Убедись, что у тебя установлен Python 3.6+**  
   Проверь в терминале:
   ```bash
   python --version
   # или
   python3 --version

2. **Скачай или клонируй проект**
    ```bash
    git clone https://github.com/your-username/frost-knight.git
    cd frost-knight
3. **Запусти игру**
    ```bash
   python main.py
4. **Следуй инструкциям в консоли**
    Используй клавиши (1,2,3) или (h,s) в блэкджеке

## Структура проекта

Heir_to_Icecrown:
**main.py -- Запускает игру.**
characters.py -- Классы персонажей (рыцарь, дракон, нежить).
combat.py -- Боевые механики (битвы с нежитью и драконом).
story.py -- Сюжетные сцены и финальные выборы.
blackjack.py -- Мини-игра "Блэкджек" против дракона.
README.md -- Этот файл.

heir-to-icecrown/
│
├── main.py                 # Точка входа. Запускает и управляет ходом игры.
│
├── characters.py           # Классы персонажей:
│                           #   • Character — базовый класс
│                           #   • Knight — главный герой (рыцарь Альянса)
│                           #   • UndeadWarrior — враг (рыцарь смерти)
│                           #   • Dragon — финальный босс (Сарторакс)
│
├── combat.py               # Боевые механики:
│                           #   • fight_undead_ambush() — засада нежити
│                           #   • fight_dragon() — битва с Сартораксом
│
├── story.py                # Сюжетные сцены:
│                           #   • scene_0_intro() — введение
│                           #   • scene_1_sword() — выбор меча Артаса
│                           #   • scene_2_betrayal() — лаборатория Келдриса
│                           #   • final_choice() — финальный выбор и концовки
│
├── blackjack.py            # Мини-игра «Кровавый Блэкджек» против дракона
│                           # (активируется в концовке 3)
│
└── README.md               # Этот файл — документация и инструкция по запуску
